# Git

## 第一次使用时（初始化并推送到远程仓库）
```bash
git init                             # 初始化 Git 仓库
git remote add origin XXX            # 添加远程仓库地址
git add .                            # 添加所有文件到暂存区
git commit -m "Your description"     # 提交修改
git push -u origin master            # 推送到远程仓库并建立追踪关系
```
## 后续开发使用（拉取-修改-提交-推送）
```bash
git pull origin master               # 从远程仓库拉取最新代码
git add .                            # 添加修改到暂存区
git commit -m "Your description"     # 提交修改
git push origin master               # 推送到远程仓库
```
