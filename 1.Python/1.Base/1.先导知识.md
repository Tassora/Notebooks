## 1. 解释器和编译器的区别
- 编译器：将整个源代码一次性翻译成机器代码，生成可执行文件，之后运行时无需源代码。例如：C、C++
- 解释器：逐行读取源代码，将其翻译成机器能理解的指令并立即执行，不生成独立的可执行文件。例如：Python

## 2. Python的注释

&emsp;&emsp;Python 提供 **单行注释** 与 **多行注释** 两种方式，主要用于说明代码逻辑、调试代码或屏蔽部分代码内容。

### 2.1 单行注释

* 使用 `#` 开头
* 注释符后面的内容在运行时会被解释器忽略

```python
# 这是一个单行注释
print("Hello, world!")  # 输出语句
```

### 2.2 多行注释（两种常用方式）

#### 方法一：连续使用多行 `#`

```python
# 这是多行注释
# 可以用于注释掉多行说明
# 或屏蔽多行代码
```

#### 方法二：使用三引号（`'''` 或 `"""`）

```python
'''
这是一个多行注释的例子
通常用于说明模块、类或函数的用途
解释器会将其视为字符串而忽略
'''
```
