变量#

1. 变量的定义#

a = 10              # 1.定义变量
print("a =", a)     # 2.打印变量
a = 20              # 3.修改变量
print("a =", a)     # 4.打印修改后的变量

a = 10
a = 20

📌 定义变量时，需注意：

  • 变量名应 见名知意，便于阅读和维护
  • 由 字母、数字 和 下划线 组成，但不能以数字开头
  • Python 变量区分大小写，如 Name 和 name 是两个不同变量

2. 变量类型#

2.1 变量类型概述#

类型 名称 示例 说明
int 整数 1, -42 用于表示整数类型的数据
float 浮点数 3.14, -0.5 表示小数或实数
bool 布尔值 True, False 用于逻辑判断，只有两个值
str 字符串 "hello", '123' 一串字符，用单引号或双引号括起来

2.2 变量类型的定义、查看#

a = 100                         # int
print(a,"的类型是", type(a))
a = 3.14                        # float
print(a,"的类型是", type(a))
a = True                        # bool
print(a,"的类型是", type(a))
a = "Hello, World!"             # str
print(a,"的类型是", type(a))

100 的类型是 <class 'int'>
3.14 的类型是 <class 'float'>
True 的类型是 <class 'bool'>
Hello, World! 的类型是 <class 'str'>

📌 提示： 多行字符串需要使用 """ 或 ''' 包裹。

2.3 变量类型转换#

a = 10.1
print(a,"的类型是", type(a))
print(int(a),"的类型是", type(int(a)))

10.1 的类型是 <class 'float'>
10 的类型是 <class 'int'>

  在 Python 中，不同类型的变量可以通过内置函数进行转换，常见的转换函数如下：

函数 作用 示例 结果
int() 转为整数 int(3.7) 3
float() 转为浮点数 float(5) 5.0
str() 转为字符串 str(100) "100"
bool() 转为布尔值 bool(0) False 


a = 10.9
print("a = 10.9, 转化后int(a) =", int(a))    # 转为整数：10
b = "123"
print("b = '123'，转化后 int(b) =", int(b))  # 字符串转整数：123
c = 0
print("c = 0, 转化后bool(c) =", bool(c))     # False，0在布尔中为假
d = 3
print("d = 3, 转化后float(d) =", float(d))   # 转为浮点数：3.0

a = 10.9, 转化后int(a) = 10
b = '123'，转化后 int(b) = 123
c = 0, 转化后bool(c) = False
d = 3, 转化后float(d) = 3.0
✅ 注意事项：
• int()、float() 等函数只能转换合法格式的字符串，例如 int("12") 可行，但 int("abc") 会报错；

• int() 对 float 类型使用，会降低精度

• bool() 中：0、空字符串 ""、空列表 []、空字典 {} 等都被视为 False；

• str() 可将任意对象转换为字符串形式。
