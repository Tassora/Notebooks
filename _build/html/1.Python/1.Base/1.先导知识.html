前言#

1. 解释器和编译器的区别#

  • 编译器：将整个源代码一次性翻译成机器代码，生成可执行文件，之后运行时无需源代码。例如：C、C++

  • 解释器：逐行读取源代码，将其翻译成机器能理解的指令并立即执行，不生成独立的可执行文件。例如：Python

区别点 解释器 编译器
执行方式 逐行解释执行 先整体编译成机器码后再执行
生成文件 不生成独立的可执行文件 生成独立的可执行文件
运行速度 较慢（每次运行都需重新解释） 较快（直接运行编译后的机器码）
调试方便性 更加方便，容易定位和修改错误 相对复杂，需重新编译

2. Python的注释#

  Python 提供 单行注释 与 多行注释 两种方式，主要用于说明代码逻辑、调试代码或屏蔽部分代码内容。

2.1 单行注释#

  • 使用 # 开头

  • 注释符后面的内容在运行时会被解释器忽略

# 这是一个单行注释
print("Hello, world!")  # 输出语句

2.2 多行注释（两种常用方式）#

方法一：连续使用多行 ##

# 这是多行注释
# 可以用于注释掉多行说明
# 或屏蔽多行代码

方法二：使用三引号（''' 或 """）#

'''
这是一个多行注释的例子
通常用于说明模块、类或函数的用途
解释器会将其视为字符串而忽略
'''
📌 快捷键： 选中指定行后使用 Ctrl + / 添加或取消注释

3. Python的输出#

  Python 使用内置的 print() 函数将数据输出到控制台，支持输出文本、变量、表达式结果等内容。

3.1 输出单个值#

print("Hello, world!")

Hello, world!

3.2 输出多个值（逗号分隔）#

a = 10
b = 20
print("a =", a, ", b =", b)

a = 10 , b = 20
📌 多个输出项之间默认用空格隔开。

3.3 换行输出 与 不换行输出#

默认换行输出：

print("第一行")
print("第二行")

第一行
第二行

不换行输出（通过 end 参数）：

print("第一行", end=" ")
print("第二行")

第一行 第二行
📌 `end` 的默认值为 "\n"，可设置为 " "、"," 等来自定义结尾符。

3.4 格式化输出（%格式）#

name = "Alice"
score = 95.5
print("姓名：%s，成绩：%.1f" % (name, score))

姓名：Alice，成绩：95.5

格式说明：

  • %s：字符串

  • %d：整数

  • %5d：整数（期望得到5位数的整数，前面补充空格）

  • %05d：整数（期望得到5位数的整数，前面补充0）

  • %f：浮点数（默认 6 位小数）

  • %.1f：浮点数，保留 1 位小数

3.5 插值表达式（f-string）#

name = "Bob"
high = 180.58
print(f"{name} 的身高是 {high:.1f} cm")

Bob 的身高是 180.5 cm

3.6 小结#

用法方式 示例 说明
基本输出 print("Hello") 输出单个值
多值输出 print("a =", a, ", b =", b) 多个值中间默认用空格分隔
不换行输出 print("A", end=" ") 设置 end 参数控制行尾字符
格式化输出 "%.2f" % 3.14159 老式格式化方式
f-string f"{name} is {age} years old" 推荐格式化方式，支持表达式嵌入

4. Python的输入#

  input() 函数用于从键盘接收用户输入，返回的内容默认是字符串类型。如果需要其他类型（如整数、浮点数），需要进行类型转换。例如：

age = input("请输入年龄：")    # 输入的内容是字符串
age = int(age)                # 转换为整数类型

📌 注意：input() 返回值始终是字符串，使用时需根据需求转换变量类型。
