运算符#

  Python 中的运算符用于执行各种运算操作，主要分为以下几类：

1. 算术运算符#

运算符 说明 示例 结果
+ 加法 3 + 2 5
- 减法 5 - 1 4
* 乘法 4 * 2 8
/ 除法 9 / 3 3.0
// 整除 9 // 4 2
% 取余 9 % 4 1
** 幂运算 2 ** 3 8

2. 赋值运算符#

运算符 说明 示例 结果
=赋值a = 5变量 a 值为 5
+=加并赋值a += 3等同于 a = a + 3
-=减并赋值a -= 2等同于 a = a - 2
*=乘并赋值a *= 4等同于 a = a * 4
/=除并赋值a /= 2等同于 a = a / 2
//=整除并赋值a //= 3等同于 a = a // 3
%=取余并赋值a %= 2等同于 a = a % 2
**=幂并赋值a **= 2等同于 a = a ** 2

3. 比较运算符#

运算符说明示例结果

!=不等于5 != 3True
>大于4 > 2True
<小于2 < 5True
>=大于等于3 >= 3True
<=小于等于1 <= 4True

4. 逻辑运算符#

运算符说明示例结果

5. 位运算符#

6. 成员运算符#

运算符说明示例结果
in是否在序列中'a' in 'apple'True

7. 身份运算符#

运算符说明示例结果
is是否为同一对象a is bTrue / False
is not是否不是同一对象a is not bTrue / False

8. 三元运算符#

<结果1> if <条件> else <结果2>

  • 如果 <条件> 为真（True），返回 <结果1>；

  • 如果 <条件> 为假（False），返回 <结果2>。

示例：

a, b = 10, 20
max_value = a if a > b else b
print("较大值是：", max_value)

较大值是： 20
