Markdown#

1. 基础语法#

1.1 标题#

  使用 # 来表示标题，# 的个数表示标题的等级。

# 这是一级标题
## 这是二级标题
### 这是三级标题

1.2 段落#

  直接输入即可达到效果

I really like using Markdown.
I really like Python

1.3 强调#

  • 斜体: 用单个星号 * 包围文本

  • 加粗: 用两个星号 ** 包围文本

*这是斜体*
**这是加粗**
***又是斜体又是加粗***

1.4 引用#

  使用>表示引用，多个引用多个>，嵌套引用就连续两个>

> 引用一句话

> 第一层引用
>> 第二层引用

1.5 分割线#

第一句话
***
第二句话

1.6 列表#

  • 有序列表

1. First item
2. Second iteem
3. Third item

  • 无序列表

- First item
- Second item
- Third item
- - Third First item
- - Third Second item
